## Description

A [Nest](https://github.com/nestjs/nest) CRUD API project that I made, following the [freeCodeCamp tutorial](https://www.youtube.com/watch?v=GHTA143_b-s), in order to learn a TypeScript-based backend.

Tech Stack:
- Nest
- Prisma
- argon2
- Docker
- pactum

## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode
$ npm run start:dev
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
